# 🎯 Todo List Manager

Java로 만든 간단한 명령줄 기반 할일 관리 애플리케이션입니다.

## ✨ 주요 기능

- ➕ 할일 추가
- 📋 할일 목록 보기
- ✅ 할일 완료 처리
- 🗑️ 할일 삭제
- 📊 통계 보기
- 💾 데이터 자동 저장 (JSON 파일)
- ⏰ 생성 시간 및 완료 시간 기록

## 🚀 실행 방법

### 필요 조건
- Java 17 이상
- Maven 3.6 이상

### 컴파일 및 실행

```bash
# 의존성 설치 및 컴파일
mvn clean compile

# 애플리케이션 실행
mvn exec:java

# 또는 JAR 파일로 빌드 후 실행
mvn clean package
java -jar target/todo-app-1.0.0.jar
```

## 📖 사용법

프로그램을 실행하면 명령어 프롬프트가 나타납니다. 다음 명령어들을 사용할 수 있습니다:

### 명령어 목록

| 명령어 | 단축키 | 설명 |
|--------|--------|------|
| `add` | `a` | 새 할일 추가 |
| `list` | `l` | 모든 할일 보기 |
| `pending` | `p` | 미완료 할일만 보기 |
| `complete` | `c` | 할일 완료 처리 |
| `delete` | `d` | 할일 삭제 |
| `stats` | `s` | 통계 보기 |
| `clear` | - | 화면 지우기 |
| `help` | `h` | 도움말 보기 |
| `exit` | `q` | 프로그램 종료 |

### 사용 예시

```
명령어를 입력하세요 > add
제목: 장보기
설명: 우유, 계란, 빵 사기
✅ 할일이 추가되었습니다: 장보기

명령어를 입력하세요 > list
📋 할일 목록:
==================================================
[ ] ID:1 - 장보기
    설명: 우유, 계란, 빵 사기
    생성: 2025-01-22 14:30
--------------------------------------------------

명령어를 입력하세요 > complete 1
✅ 할일 완료: 장보기

명령어를 입력하세요 > stats
📊 통계:
==============================
전체 할일: 1
완료된 할일: 1
미완료 할일: 0
완료율: 100.0%
```

## 💾 데이터 저장

- 모든 할일 데이터는 `todos.json` 파일에 자동으로 저장됩니다
- 프로그램을 재시작해도 데이터가 유지됩니다
- JSON 형태로 저장되어 가독성이 좋습니다

## 🛠️ 기술 스택

- **Java 17**: 현대적인 Java 기능 활용
- **Maven**: 프로젝트 관리 및 의존성 관리
- **Gson**: JSON 직렬화/역직렬화
- **JUnit 5**: 테스트 프레임워크 (확장 가능)
